# IAM 文档导航

## 事实源优先级

发生冲突时按以下顺序处理：

1. 已批准的项目立项书和设计规格。
2. 已批准的 ADR 与领域产品规格。
3. API、事件、策略和权限的代码契约。
4. 模块 README 与运行手册。
5. 示例、注释和历史文档。

发现冲突时不得自行选择有利版本；应先修订上游规格或新增 ADR。
